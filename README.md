# asp.net-core-SportsRetail

A real e-commerce application covers the most basic functions
1. user authentication
2. shopping cart
3. payment gateway
4. Search and layout.

This project was generated with asp.net core 2.1, bootstrap 4 and Stripe payment gateway.
